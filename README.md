# 🏛️ Computer Architecture & Python Projects

Core academic implementations featuring a low-level Binary-to-ASCII utility in MIPS Assembly and a high-level automated Text-to-Speech module using Python.

---

## 📁 Projects Overview

### 1. 🔢 Binary-to-ASCII Converter
* **Language:** MIPS Assembly (`.asm`)
* **Description:** A low-level programming implementation that takes binary inputs and converts them efficiently into human-readable ASCII characters. Ideal for understanding computer architecture and data manipulation at the register level.
* **Key Features:**
    * Direct register handling
    * Efficient loop and conditional structures
    * Optimized memory usage

### 2. 🗣️ Text-to-Speech (TTS) Engine
* **Language:** Python (`.py`)
* **Description:** A high-level scripting application that automates text-to-speech conversion. It provides a clean, automated layer to read out textual data effortlessly.
* **Key Features:**
    * Quick execution with minimal lag
    * Modular and reusable code structure

---

## 🛠️ Tech Stack & Tools

* **Low-Level:** MIPS Assembly, MARS Simulator
* **High-Level:** Python 3.x
* **Environment:** Pre-configured `.gitignore` included to maintain a clean repository structure.

## ⚙️ How to Run

### MIPS Assembly Project
1. Open the `Binary-to-ASCII Converter.asm` file in a MIPS simulator (like MARS).
2. Assemble the file (`F3` or Assemble button).
3. Run the program and check the output console.

### Python TTS Project
1. Run the Python script via terminal:
   ```bash
